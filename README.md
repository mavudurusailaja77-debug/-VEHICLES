# Antigravity Motors - Vehicle Web Application

A modern full-stack web application structure with a Node/Express backend and React + Tailwind CSS (v4) frontend.

## Project Structure

- `frontend/`: React application using Vite and Tailwind CSS.
- `backend/`: Node.js Express REST API server.

## Installation & Setup

Make sure you have Node.js (version 24.18.0 or later) installed.

### 1. Install Backend Dependencies

Navigate to the `backend` directory and install dependencies:
```bash
cd backend
npm install
```

### 2. Install Frontend Dependencies

Navigate to the `frontend` directory and install dependencies:
```bash
cd frontend
npm install
```

## Running the Application

For the full application to function, you need to run both the backend server and the frontend development server.

### Start Backend

In the `backend` directory, run:
```bash
npm run dev
```
The server will start on [http://localhost:5000](http://localhost:5000).

### Start Frontend

In the `frontend` directory, run:
```bash
npm run dev
```
The Vite development server will start (usually on [http://localhost:5173](http://localhost:5173)).
